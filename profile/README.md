# Murus Firewall Software — Graphical Interface for Built-in pf Firewall for System Administrators

## About Murus Firewall Software

Murus Firewall Software provides a complete graphical interface for the built-in pf (packet filter) firewall on macOS, designed for system administrators and network professionals. Murus Firewall Software exposes full power of FreeBSD pf packet filter with user-friendly controls, advanced rule creation, and live connection monitoring. Murus Firewall Software enables users to create inbound/outbound rules, manage network zones, inspect firewall logs, and whitelist/blacklist IP addresses. Murus Firewall Software offers protocol selection, port management, launch daemon control, VPN kill switch, and traffic shaping. Murus Firewall Software runs as background service with menu bar status. Murus Firewall Software is trusted by system administrators, network engineers, security professionals, and power users worldwide.
